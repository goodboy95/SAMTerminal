# 集成/系统测试清单

- **登录注册流程**：
  - 正常注册 -> 自动登录 -> 状态接口返回新用户数据。
  - 已存在用户登录 -> 跳转 /game。
- **聊天与状态同步**：
  - 发送普通文本，收到 AI 回复，状态栏心情更新。
  - 发送 `travel_to:dream-edge`，地图状态与背景变化。
- **会话绑定**：
  - 首次进入 /game 创建 sessionId，后续请求携带 sessionId 且保持同一 API。
  - 会话超时后触发新 sessionId 返回。
- **记忆回廊回忆**：
  - 点击任意记忆卡片触发 `/api/game/memory/recall`，返回回忆回复。
- **地图数据加载**：
  - /api/world/map 返回的 domains/locations 在地图正确渲染；用户解锁状态生效。
- **背包/记忆接口**：
  - /api/player/inventory、/api/player/memories 返回的数据在模态正确显示。
- **资源与上传**：
  - 上传背景/头像后返回 `/uploads/*`，页面预览正常。
- **图片复用**：
  - 相同 URL 资源仅发起一次网络请求（背景与立绘）。
- **后台管理**：
  - 管理员登录权限校验（支持多个管理员账号；非管理员应失败）。
  - 立绘保存后再次刷新能回显。
  - 星域/区域新增、修改后再请求 map 数据有更新。
  - 角色档案新增/修改后列表更新。
  - Token 监测页能读取用量并修改全局/单用户上限。
  - LLM API 池新增/编辑/删除生效，测试连接可用，token 重置后归零。
- **部署/容器**：
  - build.sh 执行成功，docker-compose 启动后前端 8090、后端 8081 正常。
